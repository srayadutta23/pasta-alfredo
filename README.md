# 🍝 Pasta Alfredo Recipe Portfolio

A premium, interactive recipe portfolio website showcasing the classic **Fettuccine Alfredo** recipe with a modern dark-themed interface and engaging user experience.

This project was **created using Antigravity AI** and then deployed by me using **GitHub** and **Netlify**.

## 🌐 Live Demo

👉 https://pasta-alfredo.netlify.app
